# JSON Schema Repository

## Available Schemas

Browse the collection of schemas available in this repository:

| Schema Name | Description                                                                                 |                  Link                  |
| :---------: | ------------------------------------------------------------------------------------------- | :------------------------------------: |
|    Zola     | Schema for Zola, A fast static site generator in a single binary with everything built-in.. | [zola.schema.json](./zola.schema.json) |

## Usage

### Zola

Copy below line in the top of your Zola config file.

```toml
#:schema https://cscnk52.github.io/json-schemas/zola.schema.json
```
